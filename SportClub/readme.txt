

    //This platform is developed on 19-08-2016//
    SportClub is complete platform for sport(cricket) lovers.
    
    Skills used: Frontend-Bootstrap,css,html,javascript
                  Backend-MSSL SERVER 2014
                  Integeration of .NET using C#  in VS2013.
                  
       Media: Visible/comfortable for all size screen.           
                  
            Functionality:
            1>create tournament:user can create tournament and add groups,corresponding to each group
             dynamically add teams.
            2>add match to existing tournament:user can add match to existing tournamnet and select group->teams
            and can also view the scoreboard of existing matches.
            3>view list of tournaments:user can view details of particular tournamnet i.e
            league board which includes points,NRR etc of each group and teams.
